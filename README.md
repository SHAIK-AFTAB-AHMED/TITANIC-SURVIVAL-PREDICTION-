# Titanic Survival Prediction Using Machine Learning Classification Models

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using supervised machine learning techniques. The problem is treated as a binary classification task where the target variable indicates whether a passenger survived or not.

The project uses the Titanic dataset from Kaggle's "Titanic - Machine Learning from Disaster" competition and applies data preprocessing, feature engineering, model training, and performance evaluation to build an effective prediction system.

## Dataset

Source: Kaggle - Titanic: Machine Learning from Disaster

Features used include:

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Fare
* Siblings/Spouses Aboard (SibSp)
* Parents/Children Aboard (Parch)
* Embarked Port

## Machine Learning Models

The following classification algorithms were implemented and compared:

* Logistic Regression
* Decision Tree
* Random Forest
